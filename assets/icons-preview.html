SVG → PNG 내보내기 안내
Capacitor 빌드엔 PNG가 필요합니다. 고른 SVG를 1024×1024 PNG로 변환 후 npx @capacitor/assets generate --iconOnly 명령으로 iOS·Android 전체 사이즈 자동 생성 가능.
간단한 방법:
  • 온라인: CloudConvert·Vectr·Figma 등에 SVG 업로드 → 1024px PNG export
  • CLI: npx svgexport icon-1-cream.svg icon.png 1024:1024 (svgexport 설치 필요)
  • Inkscape: inkscape icon-1-cream.svg --export-type=png --export-width=1024
